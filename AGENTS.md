# Repository Instructions

## Pull Requests

- Open pull requests for this repository against `MagmaMoose/git-pull-request-dashboard` with base branch `main`.
- Do not open pull requests against `AKharytonchyk/git-pull-request-dashboard` unless the user explicitly asks for that target.
- When using `gh pr create`, pass `--repo MagmaMoose/git-pull-request-dashboard --base main` explicitly.

## Git Policy

- Branch names must follow Conventional-Commit style naming: `<type>/<description>` or `<type>/<scope>/<description>`.
- Allowed types: `feat`, `fix`, `docs`, `style`, `refactor`, `perf`, `test`, `build`, `ci`, `chore`, `revert`.
- Never create or keep PR titles containing `[codex]` in any casing.
